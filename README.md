# Retail Sales Analysis Dashboard

## 📌 Project Overview
This project analyzes retail sales data using PostgreSQL and Power BI to generate business insights and interactive dashboards.

## 🛠 Tools Used
- PostgreSQL
- SQL
- Power BI
- Excel/CSV

## 📊 Key Insights
- Total Sales KPI
- Monthly Sales Trend
- Category-wise Performance
- Top Customers Analysis
